# Cadastro Para Adoção de Gatos (Vite + React) Atv 1

Projeto simples com:
- Vite + React
- React Router (rota / e /cadastro)
- Componente `FormularioCadastro` usando useState, useRef e useEffect
- CSS Modules (tema verde menta + rosa bebê)
- Ícone SVG de gato incluído

Como usar:
1. Instale dependências: `npm install`
2. Rode em desenvolvimento: `npm run dev`
3. Abra o navegador no endereço mostrado pelo Vite (normalmente http://localhost:5173)
